# Code of Conduct

## Our Pledge

We pledge to make participation in **react-confirm-lite** a respectful, harassment-free experience for everyone, regardless of experience level, gender, gender identity, sexual orientation, disability, personal appearance, body size, race, ethnicity, age, religion, or nationality.

This project is focused on building a clean, minimal, and maintainable openâ€‘source library. All interactions should support that goal.

---

## Our Standards

Examples of behavior that contributes to a positive environment:

* Being respectful and constructive in discussions
* Giving clear, actionable feedback
* Accepting criticism gracefully
* Helping others learn without condescension
* Focusing on the code, not the person

Examples of unacceptable behavior:

* Harassment, insults, or personal attacks
* Trolling or deliberately derailing discussions
* Publishing private information without consent
* Repeatedly ignoring project guidelines

---

## Scope

This Code of Conduct applies to all project spaces, including:

* GitHub issues
* Pull requests
* Discussions
* Any other community interaction related to this project

---

## Enforcement

Maintainers are responsible for enforcing this Code of Conduct.

Unacceptable behavior may result in:

* A warning
* Temporary or permanent removal from the project

The severity of the response will be determined based on the context and impact of the behavior.

---

## Reporting Issues

If you experience or witness unacceptable behavior, please report it by opening a GitHub issue or contacting the maintainer directly.

All reports will be reviewed and handled fairly and discreetly.
